利用nCov2019包分析疫情数据

Hope

2020/2/19

本教程聚焦于如何使用南方医科大学余光创老师写的nCov2019包来对疫情数据进行获取、分析与可视化。更详细的资料可参考https://github.com/GuangchuangYu/nCov2019。

包的安装

目前包没有上CRAN，只能够在GitHub上直接下载使用。安装方式如下：

remotes::install_github("GuangchuangYu/nCov2019")

包的加载

library(nCov2019)

当日数据获取（get_nCov2019）

数据获取仅仅需要一行代码：

x <- get_nCov2019()

现在，数据已经存储在x变量中。我们不妨看一下x是什么数据结构：

地图可视化

在进行可视化之前，首先要加载地图数据，这也由老师直接提供了，获取方法如下：

remotes::install_github("GuangchuangYu/chinamap")

安装后，我们来加载数据：

require(chinamap)
## Loading required package: chinamap
cn = get_map_china()
## Loading required package: sp
## Regions defined for each Polygons

现在，中国地图的信息就在cn变量里面。 目前来说，地图可视化只支持到每天，也就是只能对单一日的疫情数据进行可视化。比如要对今天的数据进行可视化：

plot(x, region='china', chinamap=cn)
## Warning: Ignoring unknown aesthetics: x, y

其中，palette参数可以改变颜色，continuous_scale可以设置变量是离散还是连续的。例子如下：

plot(x, region='china', chinamap=cn,
    continuous_scale=FALSE,
    palette='Blues')
## Warning: Ignoring unknown aesthetics: x, y

如果想要画历史数据，则需要指定日期（比如我们想看2020年2月1号的数据）：

plot(y, region='china',
     chinamap=cn, date='2020-02-01')
## Warning: Ignoring unknown aesthetics: x, y

事实上，只要有底图数据，可视化还能够细化到市级。具体教程可在一次搞定所有的疫情地图 找到。
